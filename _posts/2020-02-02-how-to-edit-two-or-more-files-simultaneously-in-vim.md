# How to edit two or more files simultaneously in VIM

This is an fast and objective tip.

Sometime I need edit two or more files simultaneously. 

That's how I do it...

---

Horizontal:

`vim -o file01 file02`

Vertical:

`vim -O file01 file02`


That's all folks.
